# PROYECTO_LENGUA_Y_LITERATURA_SEGUNDO_DE_BASICA

## Descripción
Plataforma web interactiva desarrollada para estudiantes de segundo de básica, enfocada en el Objetivo de Desarrollo Sostenible (ODS) N° 4: Educación de Calidad. El sistema utiliza Java EE, JSP y PostgreSQL para la gestión académica.

## Estructura del Proyecto
- **/src**: Código fuente de la lógica de negocio (Java).
- **/webapp**: Archivos de interfaz (JSP), estilos (CSS) y recursos multimedia.
- **RECURSOS_3D_PROYECTO_LENGUA_Y_LITERATURA.txt**: Enlaces a recursos 3D pesados (.glb) alojados externamente.

## Tecnologías Utilizadas
- **Backend**: Java EE, Apache Tomcat, JDBC.
- **Frontend**: HTML5, CSS3, Bootstrap 5, JavaScript.
- **Base de Datos**: PostgreSQL.
- **Integraciones**: H5P para actividades interactivas.

## Instrucciones de Ejecución
1. Clonar el repositorio.
2. Configurar la base de datos PostgreSQL usando el esquema del proyecto.
3. Importar el proyecto en Eclipse como un Dynamic Web Project.
4. Desplegar en el servidor Apache Tomcat.
